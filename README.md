# 제2회 천안청소년국악관현악단 정기연주회 초대장

## 카카오톡 미리보기 캐시 관련 안내

카카오톡 링크 공유 시 제목이나 이미지가 구 버전(예: "My Google AI Studio App")으로 보인다면 아래 절차를 통해 카카오톡 서버의 캐시를 직접 초기화해야 합니다.

1. **카카오 개발자 도구 접속**: [카카오 OG 캐시 초기화 페이지](https://developers.kakao.com/tool/clear/og)
2. **URL 입력**: `https://korean-traditional-music-invitation.vercel.app/`
3. **초기화 버튼 클릭**: '초기화' 버튼을 눌러 캐시를 삭제합니다.
4. **확인**: 카카오톡 앱을 완전히 종료 후 다시 실행하거나, 새로운 채팅방에 링크를 공유하여 변경된 내용을 확인합니다.

*참고: 현재 미리보기 이미지는 `/img/background.png`를 사용하도록 설정되어 있습니다.*

## 배경음악(BGM) 안내

- **자동재생**: 브라우저 정책에 따라 페이지 진입 시 자동으로 재생을 시도합니다.
- **예외 처리**: 브라우저에서 자동재생을 차단한 경우, 사용자의 첫 상호작용(터치, 클릭, 스크롤 등) 시 재생이 시작됩니다.
- **재생 표시**: BGM이 재생 중일 때만 상단 아이콘이 천천히 회전합니다.

## 프로젝트 정보
- **기술 스택**: React, Vite, Tailwind CSS, Framer Motion
- **주요 기능**: 모바일 초대장, 배경음악 자동재생 및 fallback, 스크롤 애니메이션, 카카오톡/SNS 공유 지원
